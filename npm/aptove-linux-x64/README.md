# @aptove/aptove-linux-x64

Platform-specific binary package for Linux x64. This package is automatically installed as an optional dependency of [@aptove/aptove](https://www.npmjs.com/package/@aptove/aptove) and should not be installed directly.

## Installation

```bash
npm install -g @aptove/aptove
```

See the [main package](https://www.npmjs.com/package/@aptove/aptove) for full documentation.
